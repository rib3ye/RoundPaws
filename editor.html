GITHUB SETUP
How to get a token
  1. Go to github.com/settings/tokens
  2. Click "Generate new token"
  3. Name it Round Paws Editor
  4. Under Repository access, select "Only select repositories" and pick RoundPaws
  5. Under Permissions > Repository permissions, set Contents to Read and write
  6. Click "Generate token"
  7. Copy the token and paste it below
